CSS Units

CSS units allow you to modify a website’s design, so it’s important to understand how they operate. There are three different types of CSS units

  • Absolute Units: px, pt, PC, in, cm, mm
  • Relative Units: em, rem, %, ex, ch, fr
  • Viewport Units: VW, vh, vmin, vmax

% - relative to parent element, or root element(when position set to fixed)

em - unit of measurement equal to element's font size

rem - unit of measurements equal to root's font size

Fr - fraction of a container's free space

Viewport units - start with "v" follow by what part of viewport the unit is relative to, including height, weight, minimum, and maximum.

CSS Position Property

The position property specifies the type of positioning method used for an element

  • static
  • relative
  • fixed
  • absolute
  • sticky

Elements are then positioned using the top, bottom, left, and right properties.

Static - elements are not affected by the top, bottom, left, and right properties.

Relative - element is positioned relative to it's normal position.

Fixed - element is positioned relative to viewport(stays in the sameplace even if page is scrolled)

Absolute - element is positioned relative to the nearest positioned ancestor(if no ancestor id found it uses the document body, and moves along with page)

Sticky - element toggles between relative and fixed, depending on the scroll position.

CSS Box Model

Border - is a layer of the CSS box model that sits between margin and padding. By default does not have any width( uses border property to set it).

Margin - distance away from border of element, every HTML element have four margins that you can toggle by using marign with a specific side of that element (top, right, bottom, left).
Can be set to negative to overlap elements.

Padding - distance from content of element to it's border, every HTML element has four sides that you can toggle by usingpadding property with a specific side of that element (top, right ,bottom, left)

CSS Display Property

The display property specifies the display behavior of an element.

Some of mostly used display Values

  • block
  • inline
  • inline-block
  • flex
  • grid

Block - elements takes up entire width of their parent container or browser(in case parent is missing) unless a width is set and always start on a new line.

Inline - elements appear inline with other elements and without any line breaks.
Only accept margin, padding and border but do not accept widthand height.

Inline-block - elements appear in line with other elements but accept all the box-model properties.

Flex - Displays an element as a block-level flex container

Grid - Displays an element as a block-level grid container

CSS Background

The background property is used to define the background effect on the element.

The are 5 bacground properties that affect the HTML elements:

  • color
  • image
  • repeat
  • attachment
  • position

Color - used to set the color of bacground

Image - used to set an image as a background element.By default image covers the entire element.

Repeat/ - Can be used after image to change how image is gonna be repeated.

Attachment - used to specify if th background image is fixed or scroll with the rest of the page.

Position - use to definethe initial position of the background image. By default the image is placed top-left of the webpage.(You can set the following posiitons center, top, bottom, left, right)
